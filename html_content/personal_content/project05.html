Your Love as a Gift

Game Audio Demo (Unity | C#)

Concept: A 2D interactive game where players declare their love by inviting their partner to play. Upon reaching a target score, a customizable message appears. The game branches into different sequences based on player choices ("Yes" or "No"), with dynamic audio enhancing the experience.

Audio Programming | Music & Sound Design

  • Built a core audio manager in Unity to trigger music and SFX, utilizing audio source pooling for performance optimization.
  • Created an adaptive music system that uses smooth fades to transition between character-specific themes.
  • Composed multiple tracks, each sharing a happy melody but with unique rhythms, except for the zombie character's theme, which features a minor scale for a darker tone.
  • Designed retro-inspired chiptune SFX to give the game a nostalgic feel.

This project showcases my expertise in game audio design, technical implementation, and music composition, all built from scratch in Unity.
