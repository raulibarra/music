Simple FPS & MetaSounds Demo

3D FPS Game (Unreal | C++ | Blueprints | MetaSounds)

Concept: You have been sent to a rebel base occupied by cyborgs. Kill them all in order to claim the base for the alliance...

Audio Programming | Music & Sound Design

  • Composed music and created sound design for weapons, impacts, footsteps, and death animations.
  • Developed an interactive horizontal resequencing music system using MetaSounds, with transitions synchronized to the music's tempo.
  • Created custom MetaSound nodes for random SFX playback, mimicking audio middleware features like random containers and pitch/volume variation.
  • Integrated animations with event notifiers to trigger footstep sounds and implemented dynamic mixing and ducking via custom sound classes.

Additionally, I created the gameplay logic, enemy AI, and UI using Blueprints and C++.
